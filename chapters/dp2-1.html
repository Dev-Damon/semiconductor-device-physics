다이오드는 왜 한쪽으로만 전류를 흘릴까

pn 다이오드에 전압을 걸면 전류가 흐른다 — 그런데 그 전류의 정체는 "길 잃은 소수파 캐리어가 흩어지면서(확산하면서) 만드는 흐름"입니다. 이번 장은 그 그림을 머릿속에 또렷이 그리는 시간입니다.

화살표 → 로 넘기세요

이번 장의 목표

전류의 공식을 외우기 전에, "그림"부터

앞으로 다이오드 전류식 I = Is(eqV/kT − 1) 을 유도할 텐데, 그 전에 반드시 알아야 할 세 가지가 있습니다.

이 장에서 답할 질문

  1. 다이오드를 풀 때 우리는 어떤 단순화 가정 위에 서 있는가?
  2. 순방향 전압이 걸리면 소수 캐리어는 어디로, 얼마나 들어가는가?
  3. 그 캐리어 분포는 위치에 따라 왜 지수적으로 감소하는가? (→ 확산 길이)

결론 미리보기: 다이오드 전류는 소수 캐리어의 확산으로 결정됩니다. 이 한 문장이 이번 장의 전부예요.

5초 복습

전압을 걸면 밴드가 휜다 — 평형 → 순방향

평형에서 pn 접합은 내부전위(built-in potential)가 만든 언덕을 가지고 있습니다. 전압을 걸면 이 언덕의 높이가 바뀝니다.

  • 순방향(forward): p쪽에 +전압 → p쪽 밴드가 아래로 내려감 → 언덕(=장벽)이 낮아짐 → 캐리어가 쉽게 넘어감 → 전류 흐름.
  • 역방향(reverse): p쪽에 −전압 → 언덕이 더 높아짐 → 거의 막힘.
밴드 방향 헷갈림 방지: 전자는 "밴드 다이어그램에서 아래로" 굴러가려 하고, 홀은 "위로 떠오르려" 합니다. 순방향에서 언덕이 낮아지면 양쪽 다 반대편으로 쉽게 굴러갑니다.

이게 모든 유도의 출발점입니다. 다이오드를 세 구역으로 나눠 보세요.

  • 가해준 전압 V는 100% 공핍층에서 소모된다고 본다 → 내부전위가 (Vbi − Va)로 바뀜.

중성영역에 E-field가 0이라는 건, 그 구역에서 캐리어를 미는 "전기적 추진력"이 없다는 뜻입니다. 그럼 캐리어는 무엇으로 움직일까요? 오직 농도 차이 = 확산(diffusion)뿐입니다. 이 가정 덕분에 중성영역 문제가 "확산 단 하나"로 깔끔히 줄어듭니다. 여러 요인을 한꺼번에 따질 필요 없이, 나머지 효과를 모두 0으로 꺼 두고 확산이라는 한 가지 원인만 남겨 두는 셈이에요.
해석 가정 ②~⑥

이상(ideal) 다이오드를 만드는 약속들

비축퇴(non-degenerate)도핑이 과하지 않아 페르미준위가 밴드 안. 볼츠만 근사(E−EF ≫ kT)가 성립 → 캐리어 농도식을 쓸 수 있음.
계단형 접합(step)p쪽·n쪽 도핑이 위치에 무관하게 균일. 경계에서 뚝 끊김.
공핍 근사공핍층과 중성영역의 경계가 칼처럼 딱 나뉜다고 봄. 공핍층 안에만 E-field가 존재.
저준위 주입(low-level)주입된 잉여(excess) 캐리어 양이 다수 캐리어보다 훨씬 적음 → 다수 캐리어 농도는 거의 안 변함.
공핍층 G/R 무시공핍층을 건너는 동안 재결합(R)·생성(G)이 없다고 가정 (단순화용).
정상상태(steady-state)충분히 시간이 흘러 ∂/∂t = 0. 시간에 따른 변화 없음.

실제론 다 일어나는 일이지만, 일단 끄고 시작합니다. 핵심 메커니즘을 먼저 보기 위해서요.

가정 보강

왜 "비축퇴 = 볼츠만 근사"인가

캐리어 농도는 상태밀도(DOS) × 채워질 확률(페르미-디랙) 을 밴드 전체에 적분해 구합니다. 그런데 페르미-디랙 함수는 손으로 적분하기 까다롭죠.

f(E) = 1 / ( 1 + e(E−E_F)/kT )   ≈   e−(E−E_F)/kT
E − EF 가 kT보다 충분히 클 때, 분모의 1을 무시 → 깔끔한 지수함수로 근사.
전도대(Ec)는 페르미준위보다 한참 위에 있으니, 정작 전자가 채우는 영역에서는 이 근사 오차가 거의 없습니다. 하지만 도핑이 과하면(축퇴) EF가 Ec에 바짝 붙어 근사가 깨지고, 우리가 쓰는 농도식 자체가 못 쓰게 됩니다. 그래서 "비축퇴만 다룬다 = 볼츠만 근사를 쓴다"가 같은 말이에요.
가정에서 나오는 결론

연속 방정식 → 전류는 어디서나 일정

연속 방정식은 "어떤 지점에서 캐리어가 시간·공간에 따라 어떻게 변하는가"를 적은 식입니다.

∂n/∂t = (1/q)·(∂Jn/∂x) + G − R
①플럭스 변화 ②생성 G ③재결합 R

공핍층에 가정을 적용하면:

  • 정상상태 → ∂n/∂t = 0
  • 공핍층 G/R 무시 → G − R = 0
  • 남는 건 ∂Jn/∂x = 0 → Jn, Jp 가 공핍층 안에서 위치에 무관하게 일정
전류는 다이오드를 관통해 흐르므로, 한 줄로 직렬 연결된 회로처럼 전체에서 같은 Jtotal = Jn + Jp 값을 가집니다. 그래서 Jn 하나만 알면 Jp = Jtotal − Jn 으로 나머지가 자동 결정됩니다.
기호 약속

nn0, pn0… 첨자 읽는 법

앞으로 캐리어를 부를 때 첨자 두 개를 씁니다. 규칙만 익히면 헷갈리지 않아요.

🔤 읽는 순서

n / p = 전자냐 홀이냐  ·  가운데 작은 첨자 = 어느 영역(n형/p형)이냐  ·  끝의 0 = 평형(thermal equilibrium)이냐.

nn0n형의, 평형 전자농도 = 다수 캐리어 ≈ ND
pp0p형의, 평형 홀농도 = 다수 캐리어 ≈ NA
np0p형의, 평형 전자농도 = 소수 캐리어 (작음)
pn0n형의, 평형 홀농도 = 소수 캐리어 (작음)
np (0 없음)p형의 비평형 전자농도 = 평형값 + 주입된 잉여분

질량작용법칙: 평형에서 nn0·pn0 = np0·pp0 = ni2. 소수 캐리어가 작은 이유가 여기 있습니다.

핵심 그림 ①

순방향 = 소수 캐리어를 "건너편에 쏟아붓기"

순방향에서 장벽이 낮아지면, n형의 전자가 p형으로, p형의 홀이 n형으로 건너갑니다. 건너간 쪽에서 그들은 무엇일까요?

n형의 전자는 자기 동네에선 다수파지만, p형으로 건너가는 순간 소수파(minority)가 됩니다. 마치 모국에선 다수 민족이던 사람이 이민 가면 소수 집단이 되는 것과 같아요. 다이오드 전류의 주인공은 바로 이 "건너간 손님" — 잉여 소수 캐리어(excess minority carrier) 입니다.
  • p형 중성영역: 평형 전자 np0 위에, 주입된 전자가 더해짐 → Δnp = np − np0 (잉여분).
  • n형 중성영역: 평형 홀 pn0 위에, 주입된 홀이 더해짐 → Δpn = pn − pn0.
  • 저준위 주입이므로 다수 캐리어 농도(NA, ND)는 거의 그대로.

"잉여(excess)"는 평형 대비 늘어난 양을 뜻합니다. 우리가 풀어야 할 미지수가 바로 이 Δ값들이에요.

핵심 그림 ②

주입된 캐리어는 들어갈수록 사라진다

건너온 전자는 경계(공핍층 가장자리)에서 가장 많고, 안쪽으로 들어갈수록 줄어듭니다. 왜냐하면:

경계: 잉여 ↑↑
→
확산으로 안으로

확산은 "농도 높은 곳 → 낮은 곳"으로 캐리어를 퍼뜨립니다. 안으로 가던 전자는 p형의 풍부한 홀과 만나 재결합해 소멸 → 농도가 점점 떨어지다 결국 평형값(Δn = 0)으로 수렴.

정량화

왜 "지수함수"인가 — 연속 방정식이 답한다

중성영역엔 E-field가 없으니 확산만 남습니다. 정상상태에서 잉여 전자에 대한 연속 방정식을 정리하면:

Dn · ( d2Δnp / dx2 ) − ( Δnp / τn ) = 0
① 확산항 (Dn: 확산계수)   ② 재결합항 (τn: 캐리어 수명)

양변을 Dn으로 나누면 익숙한 2계 미분방정식이 됩니다.

d2Δnp / dx2 = Δnp / Ln2     ( Ln = √( Dn · τn ) )
"2계 도함수 = 자기 자신" 꼴 → 해는 지수함수의 조합.
형태가 y″ = y/L² 이면 일반해는 y = A·e−x/L + B·e+x/L 입니다. 안으로 들어갈수록(x→∞) 캐리어는 사라져야 하니, 무한대로 커지는 e+x/L 항은 물리적으로 말이 안 돼서 버립니다(B=0). 남은 상수 A는 경계조건(경계에서의 농도)으로 정하면 끝입니다.
핵심 개념

확산 길이 Ln: 캐리어가 죽기 전 가는 거리

경계조건 두 개 — (1) 경계 x=0 에서 Δnp(0) = 주입된 값, (2) 멀리서 Δnp(∞) = 0 — 를 넣으면 깔끔한 지수 감쇠 해가 나옵니다.

Δnp(x) = Δnp(0) · e−x / L_n
경계에서 들어간 잉여 전자가, 거리 Ln 마다 e배(≈2.72배)씩 줄어든다.
Ln = √(Dn·τn) 의 의미: 확산계수 D(얼마나 빨리 퍼지나) × 수명 τ(재결합 전 평균 생존시간). 둘을 곱해 루트 → 캐리어가 살아 있는 동안 평균적으로 이동하는 거리(단위: cm). L이 길수록 더 깊이까지 캐리어가 살아 들어갑니다.
x = Ln 지점잉여 농도가 경계값의 1/e ≈ 37% 로 감소
x = 3Ln약 5% — 사실상 거의 소멸

홀도 대칭: n형에서 Δpn(x) = Δpn(0)·e−x/L_p, Lp = √(Dp·τp).

연결고리

이 분포가 곧 전류다 — 기울기를 보라

확산 전류는 농도의 기울기(dΔn/dx)에 비례합니다(픽의 법칙). 지수 분포를 미분하면 그 자체도 지수 꼴이라, 경계 x=0 에서의 기울기가 전류를 결정합니다.

Jn,확산 = q · Dn · ( dΔnp / dx ) ∝ Δnp(0) / Ln
경계에서의 기울기 ∝ (경계 농도)/(확산 길이). 주입이 많을수록, L이 짧을수록 기울기↑ → 전류↑.
다이오드 전류의 정체 = 공핍층 가장자리로 주입된 소수 캐리어가 만드는 확산 전류. 다음 장에서 "경계 농도 Δn(0)이 전압 V에 어떻게 달라지나"를 풀면, 그 유명한 지수 I-V 곡선 I = Is(eqV/kT−1) 이 자연스레 튀어나옵니다.

스스로 답해 보세요

"외부 전압은 전부 공핍층에 걸린다"는 가정이 왜 그렇게 중요한가요?
이 가정 덕분에 양쪽 중성영역의 전기장이 0이 됩니다. 그러면 중성영역에서 캐리어를 움직이는 힘은 확산밖에 안 남아, 문제가 "확산 한 가지"로 단순해집니다. 전류식 유도가 가능해지는 출발점이에요.
주입된 잉여 소수 캐리어의 농도가 왜 위치에 따라 지수적으로 줄어들까요?
중성영역의 연속 방정식이 d²Δn/dx² = Δn/L² 꼴이 되기 때문입니다. 이 미분방정식의 (발산하지 않는) 해가 e−x/L 입니다. 물리적으로는, 확산으로 안으로 퍼지면서 다수 캐리어와 재결합해 점점 소멸하기 때문이죠.
확산 길이 L = √(Dτ) 가 길다는 것은 무슨 뜻인가요?
캐리어가 재결합해 죽기 전까지 평균적으로 더 멀리 이동한다는 뜻입니다. 수명 τ가 길거나 확산이 빠를(D 큼)수록 L이 커지고, 잉여 캐리어가 더 깊숙이까지 살아남습니다.
한 장 요약

이 장의 뼈대

  • 가정: 전압은 전부 공핍층에 (중성영역 E=0) · 비축퇴(볼츠만) · 계단형 · 공핍 근사 · 저준위 주입 · 공핍층 G/R 무시 · 정상상태.
  • 결과: 연속 방정식에서 공핍층 내 Jn, Jp가 일정 → 전체 전류 Jtotal 어디서나 동일.
  • 주입: 순방향이면 소수 캐리어가 건너편 중성영역으로 주입(Δnp, Δpn), 경계에서 최대.
  • 분포: Δn(x) = Δn(0)·e−x/L, 확산 길이 L = √(Dτ) = 캐리어의 평균 생존 이동거리.
  • 전류: 경계의 농도 기울기가 곧 확산 전류 → 다이오드 전류는 소수 캐리어 확산이 결정.
한 문장: "다이오드 전류 = 공핍층 가장자리로 주입된 소수 캐리어가 지수적으로 감쇠하며 만드는 확산 전류."

다음 → 2-2: 경계 농도 Δn(0)을 전압 V로 표현(준평형·법칙 of the junction)하고, 합쳐서 이상 다이오드 전류식 I = Is(eqV/kT − 1) 을 완성합니다.
