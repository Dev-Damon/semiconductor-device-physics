게이트 전압은 어디로 가는가

한 칸짜리 전압이 어떻게 세 군데로 쪼개져 쓰이고, 그 결과 MOS 커패시터의 용량(C)이 게이트 전압에 따라 출렁이는지 — 문턱전압 VT의 정체와 C-V 곡선을 그림으로 풉니다.

이번 장의 한 문장

전압 분배 → C-V 곡선 → 비이상 보정

게이트에 전압 VG를 걸면, 그 전압은 ① 일함수 차이를 맞추는 데(φms), ② 산화막을 가로지르는 데(Vox), ③ 실리콘 표면을 휘게 하는 데(ψs) 나누어 쓰입니다. 이 "분배"를 이해하면 문턱전압과 C-V 곡선이 통째로 따라옵니다.

큰 흐름: ① 전압이 어떻게 쪼개지나 → ② 문턱전압 VT가 무엇으로 이뤄지나 → ③ 동작 모드별 용량 변화(C-V) → ④ 저주파 vs 고주파 → ⑤ 실제 소자의 비이상 효과.

직관부터 잡기

전압은 직렬로 연결된 "저금통"들에 나눠 들어간다

수도관 하나에 밸브 세 개가 직렬로 달려 있다고 생각해 보세요. 펌프 압력(VG)을 올리면 그 압력은 세 밸브에 나뉘어 걸립니다. 어느 밸브가 뻑뻑하냐에 따라 분배 비율이 달라지죠 — MOS도 똑같습니다. 세 항의 성격만 다릅니다: 일함수 차는 고정 상수, 산화막은 용량이 일정한 커패시터, 실리콘 표면은 전압에 따라 용량이 변하는 커패시터입니다.

핵심 직관: 실리콘 표면 항만 "비선형"입니다. 나머지 둘은 고정값에 가까워서, C-V 곡선의 출렁임은 거의 전부 실리콘 표면에서 나옵니다.

출발점 — 전압을 걸기 전(평형)

접합을 만들면 밴드가 먼저 휜다

금속과 반도체를 붙이면 둘의 페르미 준위(EF)가 어긋나 있습니다. 평형이 되려면 EF가 한 줄로 맞춰져야 하므로, 그 차이(일함수 차 φms)만큼 밴드가 휘어집니다. 이번엔 휘어질 곳이 산화막과 실리콘 두 군데입니다.

그림은 대표 사례 — n+ 폴리(또는 Al) 게이트 위 p형 Si, 즉 φms < 0 인 경우입니다. φms의 부호(게이트·도핑에 따라 결정)가 반대면 표면 밴드는 위로 휩니다(축적 방향).

밴드가 휜 총량 = φms = (산화막 휘어짐 Vox0) + (실리콘 표면 휘어짐 ψs0). 평형에서는 외부 전압이 0인데도 이미 이만큼 휘어 있습니다.

핵심 식 ①

전압을 걸면: 추가 휘어짐 = 외부 전압

게이트에 VG를 더 걸면 산화막과 실리콘 표면이 추가로 더 휩니다. 평형 상태(Δ=0) 기준으로 변화량만 모으면 평형 때 있던 휘어짐이 상쇄되고, 깔끔한 식이 남습니다.

VG = φms + ψs + Vox
게이트 전압 = 일함수 차 + 실리콘 표면전위(휘어짐) + 산화막 전압.

🔩 φms

금속·반도체 일함수 차. 고정 오프셋(상수). 곡선을 좌우로 미는 역할.

🌊 ψs

실리콘 표면이 휜 정도(표면전위). 모드를 결정하는 비선형 주인공.

🧱 Vox

산화막에 걸리는 전압. 평판 커패시터처럼 표면 전하량에 비례.

핵심 식 ② — 산화막 항의 정체

MOS는 그냥 평판 커패시터다

가운데가 절연체(산화막)라 전류는 못 흐르고 전기장만 전달됩니다. 그래서 게이트에 +전압을 걸면 금속 표면에 +Qm, 실리콘 쪽엔 양은 같고 부호 반대인 전하 −Qs가 마주 섭니다 (Qm = −Qs). 평판 커패시터 공식 Q = C·V를 그대로 쓰면:

Vox = Qm / Cox = − Qs / Cox
산화막 전압 = (실리콘 전하의 반대부호) ÷ 산화막 용량. Cox = εox / tox.

부호 직관: 실리콘에 음전하(−Qs)가 모이면 게이트엔 양전하(Qm)가 모입니다. 산화막 양단에 ± 전하가 마주 보니, 그 사이에 전압 Vox가 걸립니다. 전하가 많을수록 Vox도 큽니다.

p형 실리콘 위 MOS에서 게이트 전압을 키워가면 표면 상태가 세 단계로 바뀝니다.

핵심 식 ③ — 문턱전압의 부품들

VT는 "분배 식"에 모드 조건을 넣은 것

반전이 막 시작되는 순간(표면 전자농도 = 벌크 정공농도)이 문턱입니다. 이때 표면전위는 ψs = 2φF로 고정되고, 공핍층 전하 Qdep(max)가 산화막에 분배됩니다. 식에 그대로 대입하면:

VT = φms + 2φF + Qdep / Cox
문턱전압 = 일함수 차 + 표면을 반전시키는 데 필요한 휘어짐(2φF) + 공핍전하를 받치는 산화막 전압.

여기서 Qdep는 크기(절댓값)로 둔 값입니다: Qdep = qNA·Wmax = √(2 εsi q NA · 2φF). p형의 공핍전하는 물리적으로 음수이므로 표준형은 VT = VFB + 2φF − Qdep/Cox (Qdep<0), 즉 +|Qdep|/Cox와 같습니다.

  • φms — 출발선 보정(고정).
  • 2φF — 표면을 "n형처럼" 뒤집기 위해 휘어야 하는 양. 도핑이 진할수록 커짐.
  • Qdep/Cox — 그 아래 공핍층 음전하를 게이트가 받쳐주는 몫.

VT는 전압 분배 식 VG = φms + ψs + Vox에 "반전 직전" 조건을 끼워 넣은 특수해입니다. 외울 게 아니라 분배 식에서 유도되는 값입니다.

small-signal로 넘어가기

DC로 모드를 정하고, AC로 용량을 잰다

지금까지 건 게이트 전압은 DC(예: 2V로 고정). 여기에 작게 흔드는 AC small-signal(ΔVG)을 얹으면, 그에 맞춰 전하가 ΔQ만큼 찰랑입니다. 이 비율이 바로 용량입니다.

C = dQ / dVG = Q-V 곡선의 기울기
게이트 전압을 조금 흔들 때 전하가 얼마나 따라 움직이나 = small-signal 커패시턴스.

DC는 곡선 위에서 "지금 어느 위치에 있는가"(동작점)를 정하고, AC는 바로 그 위치에서의 기울기(미분)를 잽니다. 같은 소자라도 어느 DC 점에 있느냐에 따라 기울기, 즉 측정되는 C가 달라집니다 — 그래서 C-V 곡선이 됩니다.

핵심 그림 — 이상적 C-V

모드마다 용량이 달라진다

초록: 저주파 C-V (반전에서 다시 C_ox로 복귀)

  • 축적: 정공이 표면에 딱 붙어 산화막 양단에서 마주 봄 → 용량은 C = Cox (최댓값).
  • 공핍: 공핍층이 산화막과 직렬로 추가됨 → 두께가 늘수록 C 감소.
  • 반전: 반전 전하가 다시 표면에 붙으면 산화막만 보임 → C = Cox로 복귀.
1/C = 1/Cox + 1/Cdep
공핍 구간: 산화막 커패시터와 공핍층 커패시터의 직렬 합.
왜 공핍에서 줄어드나

공핍층 = 한 장 더 끼어든 커패시터

공핍 모드에선 게이트 전하 ΔQm에 대응하는 전하가 표면에서 떨어진 공핍층 끝(고정 음이온)에 생깁니다. 두 전하가 떨어져 있으니, 산화막 커패시터 뒤에 공핍층 커패시터가 직렬로 하나 더 붙은 셈입니다.

Cdep = εsi / Wdep
공핍층이 두꺼워질수록(W↑) 공핍 커패시턴스↓ → 전체 C↓.

게이트 전압을 더 키우면 공핍층이 더 깊어지므로 C는 계속 작아집니다. 반전이 시작되면 전하가 다시 표면으로 와서 더 이상 깊어지지 않습니다.

반전에서 갈리는 길 — 주파수

저주파는 복귀, 고주파는 바닥에 머문다

반전층의 전자는 벌크에서 확산해 오거나 공핍층에서 열적으로 생성됩니다. 이 과정은 시간이 걸립니다. 그래서 흔드는 신호의 주파수에 따라 반응이 달라집니다.

🐢 저주파 (LF)

전자 공급이 느린 변화를 따라잡음. 반전층이 ΔQm에 직접 대응 → 다시 C = Cox로 올라감 (그림의 초록 복귀 곡선).

🐇 고주파 (HF, 보통 100 kHz~1 MHz에서 측정)

전자가 빠른 변화를 못 따라감. 대신 공핍층 끝이 출렁이며 대응 → Cox와 Cdep(max)가 직렬 → 최솟값에서 평탄. (저주파와의 경계는 소수캐리어 생성속도·온도에 의존 — 실온 Si에서 대략 수 Hz~kHz.)

실제 소자 — 비이상 효과 ①

산화막 전하: 곡선이 통째로 옆으로 밀린다

이상적으로는 산화막 안에 전하가 없다고 가정했지만, 실제 산화막에는 공정에서 생긴 전하가 항상 존재합니다. 이 전하가 추가 밴드밴딩을 만들어, 평탄밴드를 만드는 전압 VFB가 달라집니다.

VFB = φms − Qox / Cox
이상적이면 VFB = φms. 산화막 전하 Qox가 있으면 그만큼 보정.
  • VFB가 바뀌면 VT도 같은 양만큼 이동 → C-V 곡선 전체가 좌/우로 평행이동.
  • 이동 방향은 산화막 전하의 부호에 따라 결정 (양전하면 보통 음의 방향으로).
정리: 산화막 전하는 곡선 모양은 그대로 두고, 가로축 위치만 미는 효과.

슬라이드 8과의 연결: 비이상(산화막 전하 포함) 시 VT = VFB + 2φF + Qdep/Cox 이고, 이상적이면 VFB = φms 이므로 슬라이드 8 식으로 환원됩니다.

비이상 효과 ②③④

일함수 차 · 양자효과 · 폴리 공핍

🔩 일함수 차

φms 자체가 0이 아니므로, 기준선이 옮겨져 C-V 곡선이 옆으로 밀립니다. VFB의 출발 항.

🌫️ 양자효과 (축적·반전)

캐리어(축적의 정공·반전층의 전자)의 전하 무게중심이 양자화로 경계면에서 약간 후퇴하고 양자 커패시턴스가 더해져 유효 산화막 두께(EOT)가 증가. 박막 산화막에서 축적·반전 용량 모두 Cox보다 작게 측정됨.

🧊 폴리 공핍

게이트가 금속이 아니라 폴리실리콘이면 게이트 쪽도 살짝 공핍됨 → 커패시터가 하나 더 직렬 → 반전에서 C가 더 낮아짐.

공통 원리: 전하가 경계면에서 떨어져 분포하면, 그 거리만큼의 커패시터가 직렬로 추가됩니다. 직렬이면 전체 C는 항상 더 작아집니다.

또한 공핍 모드에서도 반전 전하가 0이 아니라 조금씩 생기므로, 실제 곡선의 최저점 부근은 급격히 꺾이지 않고 부드럽게 이어집니다(미세 보정).

이해 점검

스스로 답해보기

게이트 전압 VG는 어떤 세 항으로 나뉘나요?
VG = φms(일함수 차) + ψs(실리콘 표면전위) + Vox(산화막 전압). 일함수 차와 산화막 항은 거의 고정, 표면전위만 비선형입니다.
반전 모드에서 저주파와 고주파 C-V가 다른 이유는?
반전 전자는 확산·열생성으로 공급되어 시간이 걸립니다. 저주파면 전자가 신호를 따라잡아 C가 Cox로 복귀하고, 고주파면 못 따라잡아 공핍층이 대신 반응해 Cox·Cdep 직렬의 최솟값에 머뭅니다.
산화막 안에 양전하가 있으면 C-V 곡선은 어떻게 변하나요?
VFB = φms − Qox/Cox가 바뀌어 곡선 전체가 가로로 평행이동합니다(모양은 그대로). 양전하면 보통 음의 방향으로 밀립니다.
한 장 요약

분배 → 모드 → 용량 → 보정

  • 전압 분배: VG = φms + ψs + Vox, 그리고 Vox = −Qs/Cox.
  • 문턱전압: VT = φms + 2φF + Qdep/Cox — 분배 식에 반전 조건 대입.
  • C-V: 축적·반전에서 C=Cox(최대), 공핍에서 직렬로 감소. 곡선 기울기 = small-signal C.
  • 주파수: 반전에서 저주파는 복귀, 고주파는 최솟값 유지.
  • 비이상: 산화막 전하·일함수 차는 곡선을 밀고, 양자효과·폴리 공핍은 직렬 추가로 C를 낮춤.

면접 한 줄: "MOS의 C-V 곡선은 게이트 전압이 산화막·공핍층·반전층에 어떻게 분배되느냐의 그림이고, VFB 이동(산화막 전하·일함수)과 직렬 커패시터 추가(양자·폴리 공핍)가 실제와 이상의 차이를 만든다."
